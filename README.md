# 📊 Customer Churn Prediction

## 📌 Project Overview
This project aims to analyze and predict customer churn in a telecommunications company using machine learning.

## 🎯 Objectives
- Identify factors that influence customer churn
- Build a predictive model to classify churn
- Provide business recommendations

## 📂 Dataset
Telco Customer Churn Dataset

## 🔍 Key Insights
- Customers with high monthly charges are more likely to churn
- Customers without tech support have higher churn rates

## 🤖 Models Used
- Logistic Regression
- Random Forest

## 📊 Model Performance

| Model | Accuracy | Recall (Churn) | F1 Score |
|------|--------|----------------|----------|
| Logistic Regression | 0.81 | 0.58 | 0.62 |
| Random Forest | 0.80 | 0.46 | 0.55 |

## 💡 Business Recommendations
- Optimize pricing strategy for high-paying customers
- Improve and promote tech support services
- Provide retention programs for high-risk customers

## 🔥 Key Findings
- High monthly charges increase churn risk
- Lack of tech support significantly impacts retention
- Logistic Regression performs better in detecting churn customers
